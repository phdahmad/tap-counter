# Tap Counter — عدّاد

عدّاد بسيط وأنيق يعمل على الجوال والمتصفح. تصميم بحس عربي، خطوط Fraunces، ويدعم اللمس والاهتزاز والتخصيص الكامل.

## الاستخدام

- **اضغط** على الشاشة → +1
- **اضغط مطولاً** (٠٫٨ ثانية) → إعادة تصفير العدّاد
- **زر −** في الأسفل → إنقاص
- **زر الإعدادات** (الترس) → تشغيل/إيقاف الاهتزاز، علامة +1، الواجهة الدنيا...
- **زر المظهر** (الدائرة) → تغيير لون الخلفية والنص

## المزايا

- يدعم RTL والعربية بشكل كامل
- يحفظ العدّاد والإعدادات في `localStorage`
- يعمل بدون إنترنت (بعد التحميل الأول)
- متجاوب لكل الشاشات
- ٢٠ لون للخلفية و١٠ ألوان للنص
- اهتزاز خفيف عند الضغط (على الأجهزة المدعومة)

## النشر

الملف الرئيسي هو `index.html` — صفحة واحدة مكتفية ذاتياً، تنشر على GitHub Pages مباشرة.

## التطوير

لا حاجة لأي بناء أو حزم — افتح `index.html` في المتصفح مباشرة.
